The Mechanism

How it Works.

Simple, intuitive control using standard Blender tools.

AnimTex_Controller (Empty)
1

The Controller

AnimTex creates a standard Blender Empty object called AnimTex_Controller. This invisible helper acts as the anchor for your transition.

2

Object Coordinates

The shader calculates the distance from this Empty object. As you move the Empty in 3D space, the "zero point" of the transition moves with it.

3

Total Control

Move to slide the wipe.
Rotate to angle the transition.
Scale to adjust the gradient falloff.

Technical Specifications

Complete Feature List

Everything included in version 1.0.5

Input & Transition

  • Image Mode: Blend two image textures directly.
  • Material Mode: Deep-copy full node trees from existing materials.
  • Wipe Mode: Sharp, directional transition line.
  • Blend Mode: Smooth, gradient-based mixing.
  • Reveal Mode: Transition from transparent (Alpha) to material.
